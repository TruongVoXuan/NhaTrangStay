# 🏠 NhaTrangStay

NhaTrangStay là nền tảng web hỗ trợ tìm kiếm, đăng tin và quản lý phòng trọ tại Nha Trang.

Người dùng có thể:
- Tìm kiếm phòng trọ theo khu vực
- Xem bản đồ vị trí phòng
- Đăng bài cho thuê
- Chat với AI hỗ trợ tìm phòng
- Quản lý bài đăng cá nhân
- Đánh giá và lưu phòng yêu thích

---

# 🚀 Công Nghệ Sử Dụng

## Frontend
- ReactJS
- SCSS
- React Router
- Axios
- Leaflet Map
- Lucide React Icons

## Backend
- Spring Boot
- Spring Security + JWT
- MySQL
- REST API

## Khác
- Supabase Storage
- Cloudinary
- OpenAI API

# ✨ Chức Năng Chính

## 👤 Người dùng
- Đăng ký / Đăng nhập
- Tìm kiếm phòng trọ
- Xem chi tiết bài đăng
- Lưu phòng yêu thích
- Đánh giá phòng
- Chat AI hỗ trợ tìm phòng

## 🏠 Chủ trọ
- Đăng bài cho thuê
- Upload nhiều ảnh
- Quản lý bài đăng

## 🛡️ Admin

- Khóa / mở khóa bài đăng
- Quản lý người dùng
